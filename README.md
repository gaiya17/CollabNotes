# Collab Notes App

A collaborative note-taking web application built with the MERN stack.

## Tech Stack
- MongoDB
- Express.js
- React
- Node.js
- Tailwind CSS

## Features (Planned)

- JWT Authentication
- Note CRUD
- Rich Text Editor
- Full-text Search
- Collaborator Management
- Soft Deletes
- Pagination

This project is being built as part of a Software Engineering Internship technical assessment.
